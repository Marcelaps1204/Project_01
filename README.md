# Project_01

Projeto: Previsão de Adesão ao Programa de Fidelidade de uma Companhia Aérea usando Árvores de Decisão

Descrição:
Este projeto foi desenvolvido com o objetivo de auxiliar o time comercial de uma companhia aérea a prever a probabilidade de os clientes aderirem ao programa de fidelidade da empresa. O programa oferece três tipos de cartões de fidelidade: Star, Nova e Aurora, cada um com diferentes níveis de benefícios e prêmios.

Para alcançar altas taxas de adesão e bater as metas estabelecidas, foi necessário criar um modelo preditivo capaz de determinar a probabilidade de cada cliente aderir a cada um dos cartões disponíveis. Para isso, utilizei técnicas de machine learning, em particular o algoritmo de Árvores de Decisão, implementado em Python.

O processo envolveu a análise exploratória dos dados fornecidos pelo time de marketing, que incluíam informações sobre clientes potenciais e seu histórico de interações com a companhia aérea. Em seguida, os dados foram pré-processados e divididos em conjuntos de treinamento e teste.

O modelo de Árvore de Decisão foi treinado utilizando o conjunto de treinamento, buscando encontrar padrões nos dados que pudessem prever com precisão a adesão de cada cliente a um dos cartões de fidelidade. Após o treinamento, o modelo foi avaliado utilizando o conjunto de teste para garantir sua eficácia e precisão.

Os resultados obtidos fornecerão ao time comercial uma ferramenta valiosa para direcionar suas abordagens de venda, permitindo-lhes oferecer o cartão de fidelidade com a maior probabilidade de adesão para cada cliente específico. Isso resultou em uma otimização do tempo de venda e um aumento significativo na receita, além de ajudar a alcançar as metas de adesão ao programa de fidelidade da companhia aérea.

Tecnologias Utilizadas:

Python
Biblioteca scikit-learn
Algoritmo de Árvores de Decisão - DecisionTree Classifier

Project: Prediction of Loyalty Program Enrollment for an Airline Company using Decision Trees

Description:
This project was developed with the aim of assisting the commercial team of an airline company in predicting the probability of customers enrolling in the company's loyalty program. The program offers three types of loyalty cards: Star, Nova, and Aurora, each with different levels of benefits and rewards.

To achieve high enrollment rates and meet established goals, it was necessary to create a predictive model capable of determining the probability of each customer enrolling in each of the available cards. For this purpose, machine learning techniques were used, particularly the Decision Trees algorithm, implemented in Python.

The process involved exploratory analysis of the data provided by the marketing team, which included information about potential customers and their interaction history with the airline company. Subsequently, the data was preprocessed and divided into training and testing sets.

The Decision Tree model was trained using the training set, aiming to find patterns in the data that could accurately predict each customer's enrollment in one of the loyalty cards. After training, the model was evaluated using the test set to ensure its effectiveness and accuracy.

The results obtained will provide the commercial team with a valuable tool to guide their sales approaches, allowing them to offer the loyalty card with the highest probability of enrollment to each specific customer. This resulted in optimized sales time and a significant increase in revenue, as well as helping to achieve the airline company's loyalty program enrollment goals.

Technologies Used:

Python
scikit-learn library
Decision Trees Algorithm - DecisionTree Classifier
